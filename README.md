# Z Spin

Multi-OS Hyperspin drop in replacement.


## Setup for development
### OSX

```bash
# nodejs install (brew)
$> brew update && brew upgrade brew-cask && brew cleanup && brew cask cleanup
# nodejs install (macports)
$> port install nodejs npm

# install & build app
$> make

# (optional) watch source & build
$> make watch

# run app
$> make run

# make releases
$> make release
```
